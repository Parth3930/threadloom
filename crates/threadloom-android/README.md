# threadloom-android

Android bindings and utilities for the [Threadloom](https://github.com/Parth3930/threadloom) full-stack Rust framework.

This crate provides the Android integration necessary for building cross-platform native applications with Threadloom using `tao` and `wry`.

## Features
- Android activity setup
- Wry integration for webview handling
- Threadloom framework compatibility

## Usage
This crate is generally intended to be used implicitly by the `distaff` CLI when creating and running Threadloom Android apps.
