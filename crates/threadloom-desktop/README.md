# threadloom-desktop

> Desktop app wrapper for Threadloom (coming soon).

Will provide native desktop app packaging using system WebViews. Powered by [`wry`](https://github.com/tauri-apps/wry) and [`tao`](https://github.com/tauri-apps/tao).

---

> **Note:** This crate is under active development. Expect breaking changes. Use the [`threadloom`](../threadloom) crate directly.
