# distaff

> The Threadloom CLI — new, dev, build, deploy.

`distaff` is the CLI tool for Threadloom projects. It handles project scaffolding, hot-reload dev server, production builds, and one-command Vercel deployment setup.

---

## Install

```bash
cargo install distaff
```

> **Note:** Coming soon on crates.io.

---

## Commands

| Command | Description |
|---|---|
| `distaff new <name>` | Scaffold a new Threadloom project |
| `distaff dev` | Start dev server with hot reload |
| `distaff build` | Production build |
| `distaff build --vercel` | Set up Vercel serverless deployment |
