# threadloom-core

> Core primitives for the Threadloom framework — signals, types, HTTP client.

Internal crate providing shared types and signal reactivity primitives used across all Threadloom crates. Compiles to both native (for the server) and WASM (for the browser).

---

> **Note:** This crate is an internal implementation detail. Use the [`threadloom`](../threadloom) crate directly.
